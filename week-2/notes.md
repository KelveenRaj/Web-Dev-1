# Week 2: HTML Fundamentals

## Topics Covered

1. Common HTML elements:
   - `<a>`: Links.
   - `<img>`: Images.
   - `<ul>`, `<ol>`, `<li>`: Lists.
2. HTML attributes:
   - `src`, `alt`, `href`.
3. Inline styles.

## Examples

- **Adding a Link and Image**:

```html
<a href="https://example.com">Visit Example</a>
<img src="https://via.placeholder.com/150" alt="Placeholder Image" />
```
