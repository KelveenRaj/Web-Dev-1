# Week 4: CSS Basics

## Topics Covered

1. Introduction to CSS:
   - Adding CSS to HTML (inline, internal, external).
2. Basic CSS Properties:
   - `color`, `background-color`, `font-size`, `margin`, `padding`.
3. CSS Selectors:
   - Type, class, ID selectors.

## Examples

- **Applying CSS**:

```html
<style>
  body {
    background-color: lightblue;
  }
</style>
```
