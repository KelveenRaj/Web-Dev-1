# Week 10: JavaScript Conditional Statements

## Topics Covered

1. If-Else Statements:
   - Syntax and usage.
2. Comparison Operators:
   - `===`, `!==`, `<`, `>`.
3. Logical Operators:
   - `&&`, `||`, `!`.

## Examples

- **If-Else Example**:

```javascript
const age = 15;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}
```
